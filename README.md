A beautiful, interactive birthday surprise website created with love using HTML, CSS, and JavaScript.

## ✨ Features

✅ **Password Protection** - Enter "Zafrin" to unlock the surprise
✅ **Interactive Question** - "Do you remember what date today is?"
✅ **Stunning Animation** - Floating blue balloons, twinkling stars, and fairy lights
✅ **Beautiful Design** - Blue theme with gradient backgrounds and glassmorphism
✅ **Responsive** - Works on all devices (desktop, tablet, mobile)

## 🚀 How to Access

Once GitHub Pages is enabled, the website will be live at:

```
https://zafrinhossain53-creator.github.io/tosin-birthday-surprise/
```

## 📋 Setup Instructions

1. Go to your repository settings
2. Scroll down to **Pages** section
3. Under "Source", select **main** branch
4. Click **Save**
5. Wait a few minutes for GitHub Pages to deploy
6. Your website will be live at the URL above!
